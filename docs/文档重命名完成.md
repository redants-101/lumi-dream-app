## ✅ 重命名方法

### Git 追踪的文件
使用 `git mv` 保留文件历史：
```bash
git mv docs/OLD_NAME.md docs/新名称.md
```

### 未追踪的新文件
使用 `Rename-Item`：
```powershell
Rename-Item -Path docs/OLD_NAME.md -NewName 新名称.md
```
